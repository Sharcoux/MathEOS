* Installation *
Lisez la licence du logiciel. Si vous ne comprenez pas, vous pouvez trouver une traduction non-officielle à cette adresse : http://org.rodage.com//gpl-3.0.fr.txt

Sélectionnez le répertoire d'installation

Indiquez le type d'installation à effectuer.
Si vous êtes le seul utilisateur du logiciel sur cet ordinateur, choisissez "utilisation privée". Si vous installez le logiciel sur un ordinateur partagé avec d'autres utilisateurs de MathEOS, sélectionnez "multi-utilisateur".

Indiquez sous quel nom vous souhaitez voir apparaître le logiciel dans le menu démarrer

Sélectionnez les icones que vous voulez créer dans votre environnement de travail.

Relisez le contenu de votre installation, puis attendez la fin de la copie des fichiers avant de profiter de votre logiciel
